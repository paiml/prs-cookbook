PRS Cookbook - Declarative Visualization Dashboards in Pure Rust

Overview

Binding recipes for connecting widgets and triggering actions.

Binding Types

BindingPurposeKey Config
Simple UpdateBasic widget refreshtrigger, actions
DebouncedRate-limited updatesdebounce_ms
Model InferenceML predictionsinference.* target
Chain ActionsSequential actionsMultiple actions
ConditionalState-based logiccondition

IIUR Properties

All binding recipes follow IIUR principles:

  • Isolated: Bindings are independently testable
  • Idempotent: Same trigger → same actions
  • Useful: Real-world reactive patterns
  • Reproducible: Deterministic event handling

Binding Structure

bindings:
  - trigger: "widget-id.event"   # Event source
    debounce_ms: 300             # Optional debounce
    condition: "{{ expr }}"      # Optional condition
    actions:                     # Action list
      - target: target-widget
        action: refresh
        input: "{{ value }}"
